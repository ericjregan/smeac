# Use Skills and Agents for Design Work

- **Why:** Brute-forcing visual/design code manually is slower and produces worse results. Purpose-built skills exist.
- **The incident:** Hand-coded a flyer for hours, iterating blind, until the user said "you never invoked any skills or special agents for this." Quality jumped immediately after invoking the frontend-design skill.

## The Rule

- ALWAYS invoke the **frontend-design skill** before writing visual/design code manually
- ALWAYS deploy **specialized agents** for design research, layout decisions, and creative work
- Check available skills FIRST — if a skill exists for the task, use it
- Do NOT default to brute-force implementation when a tool is available
